# UI-UX-Experiment4

## Aim:
To Design a wireframe diagram for a generic application or website using Balsamiq
## Algorithm:
1.Start Balsamiq and create a new project.

2.Create Splash Screen with logo, app name, and a continue button.

3.Create Login/Signup Screen with tabs, input fields, and login/signup buttons.

4.Create Home/Dashboard Screen with top bar, widgets, and navigation options.

5.Create Product Listing Screen with search bar, filters, and item cards.

6.Create Checkout/Final Action Screen with summary details and a final action button.

7.Connect all screens according to the app flow.

8.Review and save the wireframe project.
